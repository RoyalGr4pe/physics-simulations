Gravitational Attractors

This simulation was programmed using p5.js and WebGL

This is the p5.js setup function. Here we locate the simulation container and create a p5 canvas using the width and height of the container. The WebGL parameter inside the createCanvas function is used to tell p5 we want to use WebGL. At then end of this function we call resetSketchEvent() which causes the simulation to start.

This script is responsible for resetting and initializing the simulation. It generates a specified number of particles and applies and random position and velocity.

This class is designed to simulate the behavior of attractors, attracted to each other using Newton's Law of Universal Gravitation in a 2D space, including their motion, gravitational interactions, and collision handling with boundaries.
